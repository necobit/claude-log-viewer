# Claude Log Viewer

Claude Code のセッションログ（JSONL形式）をチャット形式で見やすく表示するブラウザベースのビューアーです。

## 特徴

- **単一HTMLファイル** - `index.html` をブラウザで開くだけで動作（サーバー不要）
- **ドラッグ&ドロップ** - ログフォルダをブラウザにD&Dするだけで読み込み
- **チャット形式表示** - ユーザー/アシスタントのメッセージを見やすく表示
- **Markdownレンダリング** - コードブロックのシンタックスハイライト対応
- **折りたたみ表示** - Thinking / Tool Use / Progress を折りたたみで表示
- **ダーク/ライトテーマ** - 切替対応（デフォルト: ダークモード）
- **セッション統計** - トークン数（input/output/cache）、メッセージ数、モデル名を表示
- **レスポンシブ対応** - モバイルでも閲覧可能

## 使い方

### 1. ブラウザで開く

```
open index.html
```

### 2. ログフォルダを読み込む

以下のいずれかの方法でログフォルダを読み込みます：

#### ドラッグ&ドロップ（推奨）

`~/.claude/projects/` 内のプロジェクトフォルダをブラウザ画面にドラッグ&ドロップします。

```
~/.claude/projects/<プロジェクト名>/
```

> **Note:** Finder で隠しフォルダを表示するには `Cmd+Shift+.` を押してください。

#### フォルダ選択ボタン

サイドバーの「フォルダを選択」ボタンから、ファイルダイアログで選択します。

> **Note:** ダイアログ内でも `Cmd+Shift+.` で隠しフォルダ（`.claude`）を表示できます。

### 3. セッションを選択

サイドバーにセッション一覧が表示されます。クリックするとチャット内容が表示されます。

## 表示内容

| 要素 | 表示方法 |
|------|---------|
| ユーザーメッセージ | 右寄せ・青系背景 |
| アシスタントメッセージ | 左寄せ・Markdownレンダリング |
| Thinking | 折りたたみ（プレビュー表示） |
| Tool Use / Result | 折りたたみ（ツール名・入力サマリー表示） |
| Sub-Agent Progress | 折りたたみ（件数表示） |
| System メッセージ | 中央寄せ・灰色 |

## セキュリティ

- **完全にローカル動作** - ログデータは一切外部に送信されません
- **ネットワーク通信** - CDN からの CSS/JS ライブラリの読み込みのみ（SRI ハッシュで検証）
- **サーバー不要** - ローカルのブラウザだけで動作します

### 使用している外部ライブラリ

| ライブラリ | 用途 | CDN |
|-----------|------|-----|
| [marked.js](https://github.com/markedjs/marked) v12.0.1 | Markdown レンダリング | cdnjs |
| [highlight.js](https://github.com/highlightjs/highlight.js) v11.9.0 | コードシンタックスハイライト | cdnjs |

## 対応ブラウザ

- Chrome / Edge（推奨）
- Safari
- Firefox

> **Note:** ドラッグ&ドロップでのフォルダ読み込みは Chrome / Edge で最も安定して動作します。
